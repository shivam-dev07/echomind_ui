# EchoMind — Flutter Meeting App UI

A Flutter mobile app UI for AI-powered meeting intelligence.

## Features

- **Recording Screen** — Beautiful animated recording interface
- **Meetings List** — View all recorded meetings with status badges
- **AI Assistant Chat** — Chat interface for querying meeting insights
- **Settings** — Profile, calendar connectors, preferences, and account management
- **Authentication** — Login / Register with email or Google Sign-In

## Run Flutter

```bash
cd meeting_app
flutter run
```

## Tech Stack

- Flutter (Dart)
- Material Design 3
- Shared Preferences for local storage
